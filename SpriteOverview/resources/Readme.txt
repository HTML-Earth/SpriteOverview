Minimum required files to get all sprites:
    Oddysee:
        d1.lvl
        e1.lvl
        f2.lvl
        l1.lvl
        r1.lvl
        s1.lvl

    Exoddus - all except:
        bm.lvl
        cr.lvl
        sv.lvl
        
Use the extractor on those .lvl files,
then run this program in the same folder.

It will copy all unique sprites into one folder and distinguish AO/AE files with the same name.

Input_DifferingHomonymousBans.txt should contain a list of all files with the same name, but different content.
